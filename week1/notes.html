Notes

  • Mobile First Design
      We need to reqire our brains to do mobile designing first.
      Mobile traffic from 2013-2014 increased by 20%. This means that more people are choosing to look at the web using mobile devices more and more. Webpages need to be fully equipped to handle this.
      Just because a website is responsive, does not mean its mobile first.
      Responsive Retrofitting
      Start Small and Scale Up
  • Understanding Users
      Be There
      Be Useful
      Be Quick
      75% of users rely on thumbs, make sites accessible from your thumb.
      Speed Matters
      Networks Matter
  • Better Mobile Menus
      Be empathetic to user needs
      Most commonly used menu items should be at the bottom
      Hamburger symbol is hindering many people, not helping.
  • Mobile Friendly Forms
      Make it simple. Make users think less.
      Ditch labels, they take up valuable space. Use placeholders instead.
      Float Labels are labels that go to the top of a field when user is typing. So the user can still see what they should be filling in.
      Don't ask for unnecessary information.
      Passwords Matter - Show passwords by default, but give the user the option.
      Numbers Matter - Space numbers accurately and use the number pad.
      Calls Matter - Link phone numbers on the website to make call easier for users.
  • Better Mobile Advertising
      Pop-Ups are the most detested thing on the web.
      Don't have big ads - instead do ads between content or at bottom of page.
      CONTENT SHOULD COME FIRST!
